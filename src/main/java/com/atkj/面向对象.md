## 状态模式

行为型模式

一个对象的行为取决于他的状态。

比如人有两种状态，开心 伤心。

1. 两种状态可以相互转换
2. 不同状态下人的行为会不同

状态模式角色：

1. Context环境类 即有状态的对象
2. State 抽象状态类 
3. ConcreteState 具体状态类
